# TubNew Desktop

## 🎬 YouTube Downloader - Versão Desktop

Baixe vídeos do YouTube em alta qualidade direto no seu computador, sem bloqueios!

## 📋 Requisitos

- Windows 10/11
- Python 3.8 ou superior ([Download aqui](https://www.python.org/downloads/))

## 🚀 Como usar

### Primeira vez:

1. Extraia todos os arquivos em uma pasta
2. Clique duas vezes em `INICIAR.bat`
3. Aguarde a instalação das dependências (só acontece na primeira vez)
4. O navegador abrirá automaticamente

### Próximas vezes:

- Apenas clique em `INICIAR.bat` novamente!

## 📁 Onde  ficam os vídeos?

Os vídeos são salvos em: `C:\Users\SeuUsuario\Downloads\TubNew\`

## ❓ Problemas comuns

**"Python não encontrado"**
- Instale o Python: https://www.python.org/downloads/
- Marque a opção "Add Python to PATH" durante a instalação

**"Erro ao baixar"**
- Verifique sua conexão com a internet
- Alguns vídeos privados podem não funcionar

## 💡 Dicas

- Funciona com vídeos normais e Shorts do YouTube
- Escolha a qualidade antes de baixar
- Downloads são salvos automaticamente
